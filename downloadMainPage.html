Clicking on a row in the table to select measurement data will display a compressed file of that measurement data. the file size displayed on the screen is the file size before decompression, and the file size after decompression is 10 to 15 times larger than the file size before decompression. for measurement data with more than 1,000,000 data items, the measurement data is divided by each 1,000,000 items.
